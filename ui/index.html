Bonjour et Bienvenue.
Ce site permet aux étudiants d'EPITA de réviser avec la technique des Flashcards. Cette technique permet de tester ses connaissances sur les matières d'EPITA. Ce site ne remplace pas les cours, il sert uniquement à les compléter. Le bouton refresh permet de télécharger les nouvelles cartes sur la session actuelle. Le bouton Go permet de sélectionner une matière puis un jeu de carte. Ensuite, une question est affichée. Si vous connaissez la réponse, appuyez sur le bouton Vert, sinon, vous pouvez afficher la réponse avec le bouton Orange.

Le bouton Vert supprime la carte du jeu actuel et vous donne une nouvelle question.
Le bouton Rouge ne supprime pas la carte. Elle vous sera redemandée plus tard.
